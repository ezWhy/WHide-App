# WHide App - Lightweight Window Stealth & Audio Controller
### Part of the W-Series by ezWhy.com

WHide App is a professional, lightweight utility designed for Windows 10 and 11 that empowers users to instantly hide running application windows from the Taskbar and Alt-Tab view while managing audio outputs. Built with an isolated protection pipeline, it ensures high performance and code security for commercial readiness.

---

## Key Features

- **Immediate Window Stealth:** Instantly removes targeted active application windows from the Windows Taskbar and Alt-Tab navigation loop, leaving zero traces on the desktop interface.
- **Audio Control for Hidden Apps:** Mute or unmute hidden applications, allowing you to easily control audio outputs without affecting the overall system sound.
- **Emergency Global Hotkeys:** Bind customized system-wide global keyboard shortcuts to instantly execute hiding routines or recall the WHide App controller window from running in the background.
- **Password Security Matrix:** Lock access to WHide App with a protective password. If the password is input incorrectly more than 5 times, a prompt will appear to restart the computer to reset the password, providing the best protection for your hidden applications.
- **Data Location & Upgrade Preservation:** All application profiles, configurations, and local databases are securely isolated inside the system sandbox directory at `%APPDATA%\ezWhy\WHideApp\`. The installer is strictly engineered to ensure that this data directory remains untouched and fully preserved across all future application updates or version overrides.

---

## 💻 System Requirements & Installation

- **OS Support:** Windows 10 / Windows 11 (64-bit architecture optimized).
- **Deployment Wizard:** Simply download and run `WHide_App_Setup_v1.4.6.exe`, review the official End-User License Agreement (EULA), and follow the standalone installer steps to deploy.

---

## 🛡️ Data Privacy & Compliance Notice

WHide App operates 100% locally on the user's local machine device. It stores user profiles, encrypted password structures, and hidden state history tables inside an isolated offline environment. The application utilizes network connections exclusively to ping the official secure ezWhy update endpoint to check for newer version releases—it does not auto-download, auto-install, or transmit user application metadata. Please utilize this utility lawfully and in strict compliance with your workplace guidelines and local legal frameworks.
